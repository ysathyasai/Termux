## Getting Started

1. **Clone the repository**:
   ```bash

   cd Cool\ Stuff
   ```

2. **Explore guides**:
   - Start with [BasicSetup.md](../General/BasicSetup.md) if new to Termux.
   - Use `ls` to browse files directly in the terminal.
